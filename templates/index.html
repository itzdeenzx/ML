Things to know !

Machine Learning is when we teach a computer to learn like how kids learn from trying things out. The computer uses data to learn how to do things and make decisions on its own.

Machine Learning can change the world by creating new technology, like self-driving cars, accurate weather forecasts, or helping with medical diagnoses. We can use Machine Learning to solve problems and do things that we couldn't do in the past.

We use Machine Learning in our everyday life, like when we use smartphones to take pictures and the phone can recognize our faces. Or when we use online maps to find directions, the computer uses Machine Learning to do these things.
